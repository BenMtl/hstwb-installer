USB Self Install for HstWB Installer
------------------------------------

Copy "kickstart", "os39", "userpackages" and "workbench"
directories to root of FAT32 formatted usb stick.

For automated installation with Cloanto Amiga Forever
installed or cd/dvd-rom iso mounted, run HstWB Image Setup:
- Windows: Double-click "hstwb_image_setup.cmd" in Windows Explorer.
- macOS: Double-click "hstwb_image_setup.command" in macOS Finder.
- Ubuntu: Double-click "HstWB Image Setup" in Ubuntu Files.
- Linux: Type "./hstwb_image_setup.sh" and press enter in terminal.

For manual installation, see readme.txt in each directory
for files to copy.
