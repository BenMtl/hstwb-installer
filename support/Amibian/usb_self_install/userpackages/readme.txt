Copy optional EAB WHDLoad Packs here
for installation of demos and games.

HstWB Installer Wiki has a "Prepare self install tutorial" at 
https://github.com/henrikstengaard/hstwb-installer/wiki/Prepare-self-install-tutorial
describing step by step how to download and prepare EAB WHDLoad Packs.

EAB WHDLoad Packs directories must be copied or moved to
following directories:
- Copy or move files from "Commodore_Amiga_-_WHDLoad_-_Demos"
  to "eab-whdload-demos".
- Copy or move files from "Commodore_Amiga_-_WHDLoad_-_Games"
  to "eab-whdload-games".
- Copy or move files from 
  "Commodore_Amiga_-_WHDLoad_-_Games_(Beta_&_Unreleased)"
  to "eab-whdload-games_beta".

Run Build Install Entries:
- Windows: Double-click "build_install_entries.cmd" in
  Windows Explorer.
- macOS: Double-click "build_install_entries.command" in
  Finder.
- Ubuntu: Double-click "Build Install Entries" in
  Ubuntu Files.
- Linux: Type "./build_install_entries.sh" and press 
  enter in terminal.
